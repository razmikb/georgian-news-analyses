# Ground News Georgia

A [Ground News](https://ground.news)-style news aggregator for Georgia (Sakartvelo). It
scrapes major Georgian-language outlets, clusters articles that cover the same event, and
shows each event with a **coverage bar** — which side of the media reported it
(pro-government, opposition-leaning, or independent). Events covered by only one side are
flagged as **blindspots**.

The goal is a simple, honest way to see *who is covering what* in the Georgian media, and —
just as importantly — what one side is ignoring.

> **Status: early development.** The scaffolding is being built in the open. Nothing is live
> yet. Follow the commits to watch it come together.

## Why

Georgian media is sharply polarized. The same day's events look completely different
depending on which outlets you read, and it's hard to tell when a story is being pushed by
one camp or quietly dropped by another. This project makes that visible: side-by-side
coverage per event, and an explicit blindspot flag when only one leaning is reporting
something.

## How it works

```
GitHub Actions (cron, every 1–2h)
  └─ Python pipeline: fetch RSS / scrape → dedupe → embed → cluster → generate event title
       └─ Supabase (Postgres + pgvector)
            └─ Next.js site on Vercel → public, bilingual (Georgian / English)
```

- **Pipeline** — Python, run on a schedule by GitHub Actions. No server to maintain.
- **Database** — Supabase (Postgres + pgvector) for article headlines and their embeddings.
- **Clustering** — headlines are embedded and compared within a rolling 48-hour window;
  similar headlines are grouped into a single event.
- **Frontend** — Next.js + Tailwind on Vercel, with a Georgian / English toggle.

Everything runs on free tiers. Target cost: **$0 / month**.

## Sources

More outlets (TV Pirveli, Interpressnews, 1TV) come in a later phase to rebalance the mix.

## On leaning labels

The per-source leaning labels are **manual editorial judgments**, stored as data so they can
be revised — not hard-coded and not generated by a model. A public methodology page
explaining who assigned each label and why will ship with the site. The aim is transparency,
not a verdict: readers see the label *and* the reasoning behind it.

## Content & copyright

Like Ground News, this project stores and displays **headlines and links only** — never full
article text. All traffic and credit flow to the original publishers.
